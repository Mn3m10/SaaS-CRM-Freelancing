This is the documentation about our SaaS CRM project 
here you gonna find all of APIs and any notes about backend

--Note : any process on all APIs you must send the token you get from singup or login api and store it in the localstorage 
--Note : the data is only on my device because i'm using my database so any request you will do will not return any data and to solve this do the following

---APIs---

=========User APIs=========
- Signup (POST) => http://localhost:5000/api/v1/auth/signup/
- Login (POST) => http://localhost:5000/api/v1/auth/login/
- Create New User (POST) => http://localhost:5000/api/v1/user
- Get All Users (GET) => http://localhost:5000/api/v1/user
- Get User By ID (GET) => http://localhost:5000/api/v1/user/(id here)
- Update User (PUT) => http://localhost:5000/api/v1/user/(id here)
- Update User Password (PUT) => http://localhost:5000/api/v1/user/pass/(id here)
- Delete User (DELETE) => http://localhost:5000/api/v1/user/(id here)
